## 一、游戏速览

你是侦探，在一列飞驰的国际列车上调查 C7 包厢的密室死亡案。死者手指有针孔，包厢门链从内部挂上，没有任何侵入痕迹。

**凶手是谁？怎么做到的？为什么？**

## 二、核心机制

### 指令表

| 指令 | 说明 |
|---|---|
| `go <场景>` | 移动：死者包厢 / 餐车 / 列车走廊 / 行李车厢 |
| `look` | 观察当前场景 |
| `look <物品>` | 调查物品（第一次） |
| `look <物品> <二次行动>` | 深入调查（不同物品有不同的二次行动名） |
| `talk <NPC>` | 列出该 NPC 可聊话题 |
| `ask <NPC> <话题>` | 询问或闲聊 |
| `combine <卡1> <卡2> [卡3]` | 组合证据卡生成推论 |
| `reasoning` | 查看推理板状态 |
| `arrange <卡id> ...` | 按顺序排列推理链 |
| `timeline` | 查看已解锁的时间线 |
| `clues` | 查看已发现线索 |
| `accuse <嫌疑人> <手法>` | 最终指控 |
| `status` | 查看整体进度 |

### 难度选择

| 难度 | 回溯 | 时间 | 特色 |
|---|---|---|---|
| 普通 normal | 7 | 120 | 完整提示，适合初见 |
| 困难 hard | 3 | 75 | 给方向不给操作 |
| 噩梦 hell | 1 | 45 | 几乎无提示，纯硬核 |

```
new_game normal   ← 推荐初见
```

## 三、完美真结局全流程

> 目标：真结局 + 隐藏结局 = **真相大白 + 月台上的故人**

### 第一步：餐车搜集线索

```
go 餐车
look
look 公告栏
look 公告栏 检查背面          → 获得「死者未出席晚餐」
look 吧台
look 高脚杯                   → 获得「门虚掩」线索
talk 商业伙伴
ask 商业伙伴 离开时门
```

### 第二步：乘务长对话

```
go 列车走廊
talk 乘务长
ask 乘务长 巡逻路线           → 获得证词
ask 乘务长 对死者印象          → ⚠️ 触发 missing_person_trigger（复仇动机！）
```

### 第三步：死者包厢调查

```
go 死者包厢
look
look 门链                     → 第一次
look 门链 测量门缝高度         → 获得「门缝高度」
look 死者右手食指              → 第一次
look 死者右手食指 检查针孔      → 获得「针孔金属光泽」
look 床单
look 床单 分析褶皱方向         → 获得「床单褶皱」
look 棕色皮箱
look 棕色皮箱 检查内层         → 获得「伪造身份证明」
look 储物间                    → ⚠️ 触发隐藏线索
```

### 第四步：行李车厢调查

```
go 行李车厢
look
look 工作台上的工具            → 第一次
look 工作台上的工具 擦拭内壁     → 获得「金属管残留」
look 年轻画家的画箱            → 第一次
look 年轻画家的画箱 对光看       → 获得「速写本线索」
look 死者的皮箱
look 死者的皮箱 检查内封       → 获得「伪造身份证明」
look 神秘女乘客的行李箱
look 神秘女乘客的行李箱 翻开底层 → 获得「褪色车站照片」⚠️ 隐藏结局关键
look 乘务员值勤表
look 乘务员值勤表 分析记录       → 获得「乘务长巡逻记录」
```

### 第五步：NPC 关键对话

```
go 餐车
talk 神秘女乘客
ask 神秘女乘客 认识死者         → ⚠️ revenge_trigger（复仇动机关键！）
ask 神秘女乘客 昨晚在哪
ask 神秘女乘客 没有登记目的地

go 列车走廊
talk 年轻画家
ask 年轻画家 速写本             → 获得「画家白天进入」+ 需先找到画箱
ask 年轻画家 听到了什么
ask 年轻画家 案发当晚

talk 列车长
ask 列车长 对死者了解           → ⚠️ 复仇动机备选路径
ask 列车长 为什么认为有人策划  → 获得匿名信
ask 列车长 匿名信               → 需先有 anonymous_letter_info
```

### 第六步：证据组合

```
combine needle_hole_metal door_gap_height metal_tube_residue
→ 生成「针刺注射入毒」推论卡

combine wife_testimony_time steward_patrol_record
→ 生成「时间线重叠矛盾」推论卡

combine wife_testimony_time forged_id_document
→ 生成「替代身份」推论卡（自动触发复仇动机）
```

### 第七步：排列推理板

解锁条件：时间线节点 ≥ 8 个 + 至少 2 组证据组合完成。

```
arrange anonymous_letter_card passenger_list_note wife_testimony_time door_gap_height needle_hole_metal metal_tube_residue painter_entered_daytime timeline_overlap_card
```

然后把两张推论卡也排进去：
```
arrange needle_injection_card
arrange timeline_overlap_card
```

### 第八步：最终指控

```
accuse 妻子 门缝针刺注射
```

## 六、结局一览

| 结局 | 条件 |
|---|---|
| ⚡ 真结局·真相大白 | 8 核心卡正确 + 针刺注射卡 + 时间线矛盾卡 + 复仇动机已知 + 正确指控 |
| ⚡ 真结局·真相大白(短) | 同上，但缺复仇动机（不显示真名） |
| 👍 好结局·部分正义 | 正确嫌疑人+手法+针刺注射卡，但缺时间线矛盾卡 |
| 📋 普通结局·悬案 | 核心卡不足 6 张或指控错误嫌疑人 |
| ❌ 坏结局一·错位指控 | 指控画家且有陷阱卡 |
| 🚂 坏结局二·凶手消失 | 时间耗尽 |
| 🚫 坏结局三·被抛下车 | 投诉次数 ≥ 2 |
| 🌙 隐藏结局追加 | 在真/好结局之上触发 |

## 七、死亡陷阱速查

| 行为 | 后果 |
|---|---|
| `look 高脚杯 尝一口`（餐车） | 中毒死亡 |
| `look 门链 手指测试`（死者包厢） | 手指被夹断 |
| `look 门链 反锁自己`（死者包厢） | 困死密室 |
| 连续追问画家 3 次 | 被画家攻击 |
| 连续追问同一 NPC 3 次 | 投诉，2 次投诉被请下车 |

**命运弃子**：第 6 次死亡触发 MCQ，答 B 恢复，其他选项永久结束。

---

## 八、小贴士

1. **先找速写本再去问画家**——不然「速写本」话题不会出现
2. **针刺注射卡生成后，画家白天进入卡就锁死了**——先拿画家卡再做三卡组合
3. **推理板排列顺序必须正确**——前一张卡必须在后一张的 `allowed_after` 列表里
4. **`combine` 有错误上限**——噩梦难度只容错 1 次，别随便试组合
5. **投诉次数跨 NPC 累积**——别逮着一个 NPC 死磕
