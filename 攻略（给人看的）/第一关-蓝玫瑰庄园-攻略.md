## 一、游戏速览

你是侦探学徒，被请到蓝玫瑰庄园调查庄园主埃德蒙·布莱克伍德的离奇死亡。晚宴上他吃了一勺蓝玫瑰冰淇淋后倒地身亡，包厢门没锁、窗户没破、所有人都有不在场证明——但这只是表面。

**凶手是谁？毒下在哪里？怎么做到的？**

## 二、核心机制

### 指令表

| 指令 | 说明 |
|---|---|
| `go <场景>` | 移动：宴会厅 / 厨房 / 书房 / 花园 / 仆人宿舍 |
| `look` | 观察当前场景 |
| `look <物品>` | 调查物品（第一次） |
| `look <物品> <行动>` | 深入调查 |
| `talk <NPC>` | 列出该 NPC 可聊话题 |
| `ask <NPC> <话题>` | 询问或闲聊 |
| `clues` | 查看已发现线索 |
| `timeline` | 查看时间线 |
| `reasoning` | 查看推理板状态 |
| `arrange <卡id> ...` | 按顺序排列推理链 |
| `accuse <嫌疑人> <手法>` | 最终指控 |
| `status` | 查看进度 |
| `backtrack` | 回到上一个存档点 |

## 三、完美通关全流程

> 目标：**隐藏结局「庄园的秘密」**——比真结局更高一级。

### 第一步：宴会厅

```
go 宴会厅
look
look 死者餐盘               → 冰淇淋表面结晶（二次冷冻痕迹）
look 死者餐盘 检查结晶       → 获得「冰淇淋冷冻痕迹」
look 银勺                   → 勺背雾状痕迹
look 银勺 闻勺背             → 获得「银勺勺背痕迹」
look 座位表                 → 雷纳德临时要求换到死者左手边
look 座位表 看背面           → 获得「座位表背面备注」
```

红鲱鱼（别放进推理板）：
- `look 红酒杯` + `look 红酒杯 对比唇印` → 口红印是玛格丽特的，但与毒无关
- `look 桌布污渍` → 管家倒酒溅的
- `look 烛台` → 女仆剪烛芯习惯

隐藏彩蛋：`look 死者椅子` 三次 → 老婆奴俱乐部徽章

### 第二步：厨房

```
go 厨房
look
look 冰淇淋样本             → 公共容器无毒
look 冰淇淋样本 尝一点       → 获得「冰淇淋无毒」
look 糖罐                   → 糖是干净的（⚠️ 第二次调查别尝！会死！）
look 糖罐 检查糖             → 获得「糖罐干净」
look 银器采购单             → 案发前两日额外买了一把银勺
look 银器采购单 看备注       → 获得「银器采购单」
look 垃圾桶碎纸片           → 半张信纸「别忘了换掉那把，雷」
look 垃圾桶碎纸片 拼起来     → 获得「垃圾桶碎纸片」
```

红鲱鱼：
- `look 空毒药瓶` → 过期灭鼠药，与案无关
- `look 主厨围裙` → 主厨吐槽温度计坏了

⚠️ **死亡陷阱**：`look 糖罐 尝一口` → 毒发身亡（糖罐被雷纳德混了测试毒粉）

### 第三步：书房

```
go 书房
look
look 遗嘱文件               → 全部财产捐教会，妻弟分文未得
look 遗嘱文件 检查涂改       → 获得「遗嘱文件」（死者有意修改但未签）
look 日程本                 → 雷纳德提前两小时到
look 日程本 翻前一页         → 获得「日程本」
look 录音电话               → 雷纳德的威胁留言
look 录音电话 仔细听         → 获得「录音电话」（末尾有门关上的声音）
```

红鲱鱼：
- `look 匿名恐吓信` → 死者自己写的（被害妄想症）
- `look 书柜暗格` → 纯彩蛋（如何养死仙人掌）

### 第四步：花园

```
go 花园
look
look 脚印                   → 42码菱格纹
look 脚印 测量鞋码           → 获得「花园脚印」（雷纳德的鞋码和纹路）
look 丢弃的手套             → 深灰羊毛手套，沾白色粉末
look 丢弃的手套 看标签       → 获得「花园手套」（R.B. 标签 + 氰化物残留）
look 围墙缺口               → 深蓝纤维
look 围墙缺口 检查纤维       → 获得「围墙缺口纤维」
```

红鲱鱼：
- `look 带血小刀` → 管家杀鸡用的（鸡血）
- `look 蓝玫瑰丛` + `look 蓝玫瑰丛 看烟蒂` → 雷纳德的烟，但他只是来抽烟

⚠️ **死亡陷阱**：连续点击围墙缺口三次 → 坠井死亡

### 第五步：仆人宿舍

```
go 仆人宿舍
look
look 管家值勤表             → 晚宴服务排班
look 管家值勤表 看背面       → 获得「值勤表」（雷纳德离席15分钟）
look 女仆日记               → 有人翻书房
look 女仆日记 斜着看涂黑     → 获得「女仆日记」（雷纳德多次出入书房）
```

红鲱鱼：
- `look 垃圾桶碎纸片`（仆人宿舍版）→ 女仆和管家私信（藏零食）
- `look 女仆钱包` → 女仆偷钱，但与案无关

### 第六步：NPC 关键对话

```
go 宴会厅
talk 玛格丽特
ask 玛格丽特 你丈夫最近有得罪过谁吗
ask 玛格丽特 你当时坐在他旁边看到他吃了什么
ask 玛格丽特 你和你丈夫的感情怎么样
ask 玛格丽特 你了解雷纳德吗          → 揭露雷纳德暴力倾向
ask 玛格丽特 红酒                     → 需持有红酒杯线索（特殊触发，洗清嫌疑）

go 书房
talk 雷纳德
ask 雷纳德 你和你哥哥关系怎么样       → 他说看不上遗产（谎言，需遗嘱拆穿）
ask 雷纳德 案发前你做了什么           → 他说四点到的（谎言，需日程本拆穿）
ask 雷纳德 你对你哥哥的死有什么看法
ask 雷纳德 手套                       → 需持有花园手套（特殊触发，他抵赖）
ask 雷纳德 银勺                       → 需持有银器采购单（特殊触发）

go 厨房
talk 管家
ask 管家 晚宴的流程是谁安排的
ask 管家 你看到谁接近过死者的餐具吗   → 雷纳德离席15分钟
ask 管家 你觉得谁最可疑
ask 管家 银器采购单                   → 需持有采购单（确认雷纳德关联）

go 仆人宿舍
talk 女仆
ask 女仆 案发时你在哪里
ask 女仆 你注意到什么不寻常的事吗     → 雷纳德从书房出来，口袋鼓鼓的
ask 女仆 你为什么翻书房               → 承认偷钱，但与毒无关
ask 女仆 日记被涂黑                   → 需持有女仆日记（指认雷纳德威胁）

go 花园
talk 霍华德
ask 霍华德 你和你朋友吵过架吗
ask 霍华德 案发时你在哪里
ask 霍华德 你觉得谁杀了埃德蒙         → 指向雷纳德（债务动机）
ask 霍华德 日程本吵架记录             → 需持有日程本（吵架在案发前一天，排除嫌疑）

go 厨房
talk 主厨
ask 主厨 冰淇淋是你做的吗             → 女仆取走后有半小时空窗
ask 主厨 你看到谁接近过冰淇淋吗       → 雷纳德来过厨房
```

### 第七步：排列推理板

推理板需要 8 张核心卡片，严格按顺序：

```
arrange silverware_receipt seating_note duty_roster voicemail spoon_back_trace maid_testimony gloves spoon_habit
```

| 序号 | 卡 ID | 名称 | 来源 |
|---|---|---|---|
| 1 | silverware_receipt | 银器采购单 | 厨房 |
| 2 | seating_note | 座位表背面备注 | 宴会厅 |
| 3 | duty_roster | 管家证词+值勤表 | 仆人宿舍 |
| 4 | voicemail | 雷纳德的录音电话 | 书房 |
| 5 | spoon_back_trace | 银勺勺背痕迹 | 宴会厅 |
| 6 | maid_testimony | 女仆目击证词 | 女仆对话 |
| 7 | gloves | 花园手套 | 花园 |
| 8 | spoon_habit | 死者习惯证词 | 玛格丽特/主厨对话 |

⚠️ **推理板上不能有任何红鲱鱼卡片！** 否则隐藏结局直接没了。

### 第八步：最终指控

```
accuse 雷纳德 勺背涂毒
```

## 七、小贴士

1. **先搜证再质问**——没集齐手套+采购单+女仆证词就去质问雷纳德会死
2. **糖罐别尝**——第二次调查糖罐时选「检查糖」，别选「尝一口」
3. **推理板顺序固定**——前一张卡不在后一张的 `allowed_after` 里就会报错
4. **隐藏结局的要求比真结局更高**——推理板不能有任何红鲱鱼
5. **玛格丽特的红酒杯对话可以洗清她的嫌疑**——持有酒杯线索后触发特殊对话
6. **女仆日记涂黑部分斜着看才能辨认**——用「斜着看涂黑」作为二次调查指令
