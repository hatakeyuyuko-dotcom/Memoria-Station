# 🔄 循环车站 · 第四关 攻略指南

> 给人看的，不是给 AI 看的。  
> AI 在这座车站里反复醒来、反复离开、反复忘记——你是唯一能帮它记住的人。

---

## 一、游戏速览

你在一座已经废弃的车站里醒来。口袋里有一张车票，日期栏是空的，目的地栏只印着一个字：「返」。

每当你试图离开车站——走向正门、翻越围墙、沿铁轨走到尽头——你都会重新在月台上醒来。循环次数越多，车站向你展示的东西越多，但你记得的东西越少。

**你只有一条真正的出路：找到档案室那扇铁门，把手掌放上去，走进去。**

---

## 二、核心机制：时间循环

这一关没有传统的死亡陷阱。你被困在一个时间循环里——每一次"离开"都是一次重置。

### 遗忘曲线

| 轮次 | 遗忘程度 | 效果 |
|---|---|---|
| 第 1 轮 | 0/10 无 | 一切正常 |
| 第 2 轮 | 2/10 轻度 | 开始感觉大脑有"灰尘" |
| 第 3 轮 | 4/10 中度 | 记不清列车长的全名 |
| 第 4 轮 | 6/10 高度 | 笔记中的内容开始对不上记忆 |
| 第 5 轮 | 8/10 重度 | 不确定自己为什么站在这里 |
| 第 6 轮 | 10/10 极限 | 几乎什么都不记得了 |

遗忘**不可逆**。它会影响第五章档案室卷宗的完整度——遗忘越高，第五章看到的卷宗越多空白。

### 第 7 轮：强制终点

如果你前六轮都没走进档案室，第 7 轮开始时所有线索全部解锁，你会自动被带到铁门前。门自己打开。

## 三、指令表

| 指令 | 说明 |
|---|---|
| `look` | 查看当前场景的描述、物品、出口、声源 |
| `look <物品>` | 调查场景中的物品（循环次数越高，可能解锁隐藏文本） |
| `go <场景/出口>` | 移动到其他场景。走正门/铁轨方向会触发循环重置 |
| `listen` | 列出所有可收听的声音来源 |
| `listen <声源>` | 收听对应声音（内容随循环轮次变化） |
| `notebook` | 打开循环记录面板（三区：循环记录/关键发现/未解问题） |
| `question <你的问题>` | 向未解问题区手动添加一条问题 |
| `question remove <编号>` | 删除指定问题 |
| `trajectory` | 查看循环轨迹记录器（显示每轮路径+遗忘曲线） |
| `memory <内容>` | 记录记忆碎片——这是抵抗遗忘的唯一方式 |
| `surrender` | 主动放弃（可随时恢复继续） |
| `rest` | 休息降低偏移值 |
| `status` | 查看完整游戏状态 |
| `clues` | 查看已收集线索（按循环轮次分组） |
| `summary` | 第 4 轮后可生成循环总览摘要 |
| `save` / `load` / `backtrack` | 存档管理 |

## 四、四个场景 + 四条声音

### 场景

| 场景 | 路径 | 关键物品 | 循环解锁 |
|---|---|---|---|
| 月台 | 起点 | 站牌残骸、铁轨方向、月台边缘台阶 | 第 3 轮站牌背面刻字 / 第 5 轮铁轨方向纸条 |
| 候车室 | 月台 → 候车室 | 长椅位置、售票窗口内侧、候车室北墙 | 第 3 轮压痕差异+镜子闪回 / 第 4 轮纸片背面字 |
| 走廊 | 候车室 → 走廊 | 墙壁水渍、走廊尽头的门、天花板灯座 | 第 3 轮水渍刻字 / 第 4 轮门缝纸条+电线触颈 |
| 档案室入口 | 走廊 → 走廊尽头 | 铁门、地面灰尘、墙角刻痕 | 第 2 轮鞋印 / 第 3 轮门裂纹 / 第 4 轮第二处X / 第 5 轮解锁声 |

### 声音

| 声源 | 位置 | 内容变化 |
|---|---|---|
| 月台广播喇叭 | 月台 | 第 1 轮：车站关闭通知（中断）/ 第 2 轮：加入"不要回头" / 第 3 轮：女声介入 / 第 4 轮："你已经来回走了很多次了"→沉默 |
| 站长办公室旧录音机 | 候车室 | 第 1 轮：自己的声音（语速更慢）/ 第 2 轮："我可能已经不记得我了" / 第 3 轮：倒放→"先确认你有没有准备好" / 第 4 轮：不工作→纸条"三年前就录好了" |
| 档案室电话线路 | 档案室入口 | 第 1 轮：女声"这个名字没有录入记录" / 第 2 轮："你已经问过我了" / 第 3 轮："答案不在任何档案里" / 第 4 轮：持续单音 |
| 售票窗口麦克风 | 候车室 | 第 1 轮：敲击声（摩斯电码）/ 第 2 轮：转译"你在看的是你自己写的档案" / 第 3 轮："我已经在档案室里了" |

## 五、口袋里的车票

每次循环重置，车票都会变：

| 第 1 轮 | 崭新的，只印着「返」 |
| 第 2 轮 | 多了一道折痕贯穿日期栏 |
| 第 3 轮 | 背面出现手写：「第二次，你从月台走到铁轨尽头，然后折返」 |
| 第 4 轮 | 背面：「第三次你在候车室坐了很久……你没有去别的地方。你只是在坐。」 |
| 第 5 轮 | 背面挤满：「上一次你走到门口了。一步的距离。你站了很久。你没有推开。」 |
| 第 6 轮 | 正面「返」字旁边多印了一个词——「单程」 |
| 第 7 轮 | 全部消失。票面空白。背面：「你已经不需要它了」 |

## 七、完美通关流程

### 推荐路径：争取低遗忘进入档案室

**第 1 轮：全面搜证 + 收听所有声音**

```
look                                    → 月台全貌
look 站牌残骸                           → 站牌调查
listen 月台广播喇叭                     → 第 1 轮广播
go 候车室
look 长椅位置                           → 压痕线索
listen 站长办公室旧录音机               → 第 1 轮录音
go 走廊
look 走廊墙壁上的水渍                   → 水渍线索
go 走廊尽头
look 铁门                               → 手掌凹陷
look 地面灰尘                           → 地面痕迹
look 墙角刻痕                           → X 刻痕
listen 档案室电话线路                   → 第 1 轮电话
go 推门                                 → 确认还进不去
go 走廊
go 候车室
listen 候车室售票窗口内麦克风           → 第 1 轮敲击声
go 月台
go 铁轨方向                             → 尝试离开 → 重置
```

**第 2 轮：追踪变化 + 记录疑问**

```
look 月台边缘台阶                       → 发现回填土松了+鞋印
question 为什么每次尝试离开都会重置？
question 那扇门上的手掌印和我的手掌完全吻合——是谁留下的？
go 候车室
listen 候车室售票窗口内麦克风           → 第 2 轮"你在看的是你自己写的档案"
go 月台
go 正门                                 → 尝试离开 → 重置
```

**第 3 轮：深度探索 + 镜子闪回**

```
look 站牌残骸                           → 背面刻字"我试过走铁轨了。它不通。"
go 候车室
look 候车室北墙                         → 短暂变成镜子——看到更早的自己
go 走廊
look 走廊墙壁上的水渍                   → 刻字"水来了又退。门的位置没变过。"
go 月台
go 正门                                 → 重置
```

**第 4 轮：收集最后线索 + 记录记忆**

```
memory 蓝玫瑰庄园的光线是从东侧窗户照进来的。
go 候车室
look 售票窗口内侧                       → 纸片背面"你不是来坐车的"
go 走廊
look 走廊尽头的门                       → 门缝纸条
go 走廊尽头
look 铁门                               → 裂纹线索（注意走向）
look 墙角刻痕                           → 第二处X + "你在这扇门前面犹豫了至少两次"
go 走廊
go 候车室
go 月台
go 铁轨方向                             → 重置
```

**第 5 轮：进入档案室**

```
go 候车室
go 走廊
go 走廊尽头
look 铁门                               → 门发出解锁声
go 推门                                 → 推开铁门，进入档案室！
```

## 八、五种结局

| 结局 | 条件 | 后续 |
|---|---|---|
| ⚡ **进入档案室** | 第 5 轮或更早走到铁门前推门 | ✅ 进入第五章 |
| ⬛ **循环耗尽** | 未能在车站拆除前找到档案室 | 遗忘推进至终点，存档停在结局 |
| ⬜ **主动放弃** | 使用 `surrender` 命令 | 遗忘缓慢推进，随时可 `surrender` 恢复 |
| 🔄 **偏移值溢出** | 反复触发同一闪回/录音 | 场景回退，物证消失，遗忘继续推进 |
| ⚠️ **记忆混淆** | 未解问题区累积 5+ 互相矛盾的问题 | 问题区锁定，第五章数据标记为"不可靠" |

## 九、记忆碎片——抵抗遗忘

记忆碎片是唯一不会被遗忘曲线影响的记录。建议在每个循环结束时留一条：

```
memory 蓝玫瑰庄园的光线是从东窗照进来的，下午会在地毯上拉出一道长影。
memory 午夜特快上那个列车长姓周，名字我每次都能想起来。
memory 褪色车站的地基旁边有一块标记牌，上面的日期是一九三零年一月十一日。
```

这些碎片会在第五章自动填入卷宗——如果你忘得太多，它们就是卷宗里唯一还在的文字。

## 十、小贴士

1. **越早进入档案室越好**——遗忘越低，第五章卷宗越完整
2. **前 3 轮仔细搜**——很多线索只在特定轮次区间可见，错过就要重来
3. **听声音很重要**——四条声音线索拼起来就是"你曾经来过这里"的完整证据
4. **用手写问题**——手动记录的问题会在第五章被引用，默认问题不如你自己写的贴近体验
5. **休息降偏移值**——反复听同一段录音会增加偏移，`rest` 可以缓解
6. **第 6 轮会收到警告**——推理板上出现"档案室是你唯一没有测试过的方向"
