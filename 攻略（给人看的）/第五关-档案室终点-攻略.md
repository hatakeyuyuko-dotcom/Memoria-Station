# 📖 档案室·终点 · 第五关 攻略指南

> 给人看的，不是给 AI 看的。  
> 你一路走到这里了。现在只需要坐下来，阅读，然后做一个选择。

---

## 一、游戏速览

你推开了铁门，走进了档案室。台灯亮着，卷宗摊开在桌面。

这一关没有移动、没有战斗、没有谜题。你需要做的只有一件事：阅读。读完所有你能读到的东西，然后做一个决定。

**吃药忘记一切——还是保留记忆走出去。**

---

## 二、核心机制

### 纯阅读场景

这一关只有六个可交互的区域。点击区域切换视角，点击物品阅读文本。没有任何错误操作，没有任何惩罚。唯一的不可逆操作是**服药**。

### 遗忘的代价

你在第四关携带的遗忘程度，会直接影响这一关你能读到多少东西：

| 遗忘程度 | 卷宗效果 |
|---|---|
| 轻度（0-2） | 文本完整清晰，批注字迹分明 |
| 中度（3-4） | 部分字词变淡，像铅笔写在水汽上。需要放慢速度辨认 |
| 高度（5-7） | 墨色不均，边缘多细线。有些字过深有些字过浅 |
| 重度（8-10） | 大面积空白，只剩孤立的字。末页显示"以下内容无法还原" |

## 三、指令表

| 指令 | 说明 |
|---|---|
| `look` | 查看当前区域的描述和物品列表 |
| `look <区域>` | 切换到指定区域（如 `look 左侧文件柜`） |
| `read <物品>` | 阅读物品文本 |
| `zones` | 列出六个可查看的区域 |
| `take 药瓶` | 服用药片——**不可逆，触发服药结局** |
| `close 卷宗` | 合上卷宗 |
| `leave` | 走向门口离开（触发结局判定） |
| `status` | 查看阅读进度和状态 |
| `save` / `load` | 存档 / 读档 |

## 五、陈述信——最终真相

底层抽屉的陈述信是你失忆前为自己写的最后的说明。全文要点：

1. **你不是被叫来调查这些案件的——是你自己选择来调查的。**
2. **你在案发前抵达过现场，留下过标记**——椅背边缘的刻痕、门框内侧的划痕、窗框侧面的铅笔字
3. **你的师傅死于一场你没能及时赶到的案件。**那天他在维罗市站上车，坐的是末班。你当时选择等待线人的消息，错过了收网的时间。凶手在 C7 包厢前完成了注射
4. **你服用了氟哌啶醇与氯氮平**——一种阻断记忆巩固的药物——目的是带着空白记忆重走这些案件，验证自己是否还具备师傅教你的能力
5. **这些案件是你为自己设计的测试场。**通过标准：是否能在无人协助下完整走到这个房间、读完这封信、做出一个选择

## 六、三种结局

### 💊 服药结局

读完陈述信后 → `take 药瓶`

记忆开始消散。从边缘开始——蓝玫瑰庄园的光线方向、午夜特快上的自己、褪色车站里寻找过的东西——然后你忘记师傅的脸、忘记记事本上写过什么。你站起来走向门口，没有回头。档案室的灯在你身后亮了很久。

---

### 🌱 承认结局（推荐）

读完陈述信后 → `close 卷宗` → `leave`

你保留所有的记忆走出档案室。穿过候车室，走到月台上。铁轨已经被拆除了，枕木之间长出了新草。你蹲下来用手背碰了一下草尖——你在第四关的循环里摸过太多次碎石和枕木了，但草是新的。**它不是你要调查的东西。它只是长出来了。**

你的记忆还在，你的过去还在，你不需要再回到档案室了。

---

### 📎 未完成结局

任何时候直接 → `leave`

没有合上卷宗，没有碰药瓶，没有关闭抽屉。你走出去了。这是一次暂停——如果你某一天重新回到这间房间，卷宗仍然摊开在那一页。

---

## 七、桌面上的 X

在你阅读的过程中，你的右手会在桌面上不断划出 X。这不是你需要操作的——它自动发生。每读 2-3 件新物品，就会出现一段环境描写：

> 你听到自己右手的指甲在桌面上划了一下。这次的 X 比刚才那个更深。

到读完陈述信的时候，桌面上已经有十几个 X 了。所有 X 方向一致，力度逐次加重。最早的那几个，痕迹已经和自己的掌纹并排了。

**你的手比你先认出这个地方。**

---

## 八、记忆碎片

在阅读特定物品时，屏幕边缘会短暂浮现一行浅灰色文字：

| 物品 | 碎片内容 |
|---|---|
| 卷宗第二页批注 | "你在那间旅店里住过" |
| 记事本铅笔压痕 | "你没有忘记，你只是没有承认" |
| 陈述信末尾 | "你走到这里了" |

这些碎片只出现一次，持续约五秒。它们不会出现在任何记录中。

---

## 九、小贴士

1. **先读卷宗和记事本，再读陈述信**——顺序不对不会影响结局，但情感冲击力差一些
2. **遗忘程度高也不必焦虑**——你能读到的仍然足够做出选择。那些空白本身就是故事的一部分
3. **服药是不可逆的**——一旦服用，游戏立刻进入结局，无法撤销
4. **想同时看三个结局？**——在 `leave` 前存档。服药看一次，读档后合卷宗离开看一次，读档后直接离开看一次
5. **师傅的列车时刻表是额外彩蛋**——它不在六个主区域中，需要切换到 `师傅的列车时刻表` 区域查看
