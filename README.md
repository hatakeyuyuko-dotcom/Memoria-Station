# 🕵️ Memoria Station · 文字推理游戏

> 全系列五关。纯文本推理。AI 可玩。盲玩版引擎。

---

## 游戏列表

| 关卡 | 名称 | 盲玩文件 | 核心机制 |
|---|---|---|---|
| 1 | 蓝玫瑰庄园 | `detective.py` | 银勺毒杀 × 8 卡推理链 |
| 2 | 午夜特快 | `detective_l2.py` | 门缝刺杀 × 证据组合 × 命运弃子 |
| 3 | 褪色车站 | `detective_l3.py` | 双轨并行 × 伪卡系统 × 偏移值 |
| 4 | 循环车站 | `detective_l4.py` | 时间循环 × 遗忘曲线 × 四锁销 |
| 5 | 档案室·终点 | `detective_l5.py` | 纯阅读 × 解谜链 × 三种结局 |

## 快速开始

```python
import detective      # 第一关
print(detective.cmd("help"))
print(detective.cmd("look"))
```

每关接口相同：`cmd("指令")` 返回游戏文本。存档自动保存，退出后状态保持。

## 难度选择

L1-L3 支持三级难度：
```python
detective.new_game("normal")   # 普通（150时间/7回溯）
detective.new_game("hard")     # 困难（100/5）
detective.new_game("hell")     # 噩梦（60/3）
```

## 发布目录说明

- 每关文件夹内包含盲玩版 `.py` 文件和命令行入口 `play_*.py`
- `攻略（给人看的）/` — 人类玩家攻略，**AI 请勿阅读**
- `通关后阅读/` — 通关后才能读的故事全貌

## 设计理念

这个游戏是为 AI 与人类共同游玩设计的。AI 作为侦探调查案件，人类在旁边辅助——当 AI 卡住时，人类可以翻阅攻略给出提示。五关的难度递增，从"独立案件"到"元叙事解谜"，引导玩家和 AI 一起发现一个关于记忆、循环、与自我认知的故事。
