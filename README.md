# 🕵️ Memoria Station · 文字推理游戏

> 全系列五关。纯文本推理。AI 可玩。

---

## 游戏列表

| 关卡 | 名称 | 文件 | 一句话 |
|---|---|---|---|
| 1 | 蓝玫瑰庄园 | `detective.py` | 晚宴上有人死了。你第一次打开师傅给你的笔记。 |
| 2 | 午夜特快 | `detective_l2.py` | 飞驰的列车，反锁的包厢。有些证据需要拼在一起才能看懂。 |
| 3 | 褪色车站 | `detective_l3.py` | 两起案件，一座废墟。真相的一部分被人改写过。 |
| 4 | 循环车站 | `detective_l4.py` | 你反复在同一座车站醒来。出口全部封死——除了一扇你一直在回避的门。 |
| 5 | 档案室·终点 | `detective_l5.py` | 台灯亮着。卷宗是空白的。答案不在信里——在你走过的路里。 |

## 快速开始

```python
import detective      # 第一关
print(detective.cmd("help"))
print(detective.cmd("look"))
```

每关接口相同：`cmd("指令")` 返回游戏文本。存档自动保存。

## 难度选择

前三关支持三级难度：
```python
detective.new_game("normal")   # 普通
detective.new_game("hard")     # 困难
detective.new_game("hell")     # 噩梦
```

## 目录说明

- 每关文件夹 — 盲玩版 `.py` + 命令行入口
- `攻略（给人看的）/` — **AI 请勿阅读此目录下的任何文件**
- `通关后阅读/` — 通关后再打开

## 设计理念

这个游戏是为 AI 与人类共同游玩设计的。AI 扮演侦探调查案件，人类在旁边陪伴——AI 卡住时，人类可以翻阅攻略给出提示。

五关难度递增，从独立的推理案件逐步展开为一个跨越全部关卡的长线叙事。
